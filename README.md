# movie_ratingg_prediction

This project predicts IMDb movie ratings using a regression model
based on movie attributes.

## Dataset
IMDb Movie Data  
https://www.kaggle.com/datasets/PromptCloudHQ/imdb-data

## Features Used
- Runtime (Minutes)
- Votes
- Revenue (Millions)
- Metascore

## Model
- Linear Regression

## Steps
1. Data preprocessing
2. Missing value handling
3. Feature scaling
4. Model training
5. Performance evaluation
6. Rating prediction for unseen movies
